Данное приложение представляет собой базовый плеер мп3
Для добавления музыки выберите папку в которой будет музыка
Для добавления плейлиста просто введите название
Для удаления плейлиста или трека используйте двойной ЛКМ по нужному элементу
МП3 плеер поддерживает откат в истории прослушивания
То есть вы можете вернуться к треку который слушали в самом начале 
Для этого достаточно двойного ЛКМ по виджету слева от виджета старт стоп, это откатит вас на предыдущий трек
Музыка в папках music и music1
